# Kirby Vite

> Kirby plugin that handles development and production mode of Vite.

## Options

### entry

Default: `'index.js'`

### server.host

Default: `localhost`

### server.port

Default: `3000`

### server.https

Default: `false`

### server.base

Default: `/`

### build.rootDir

Default: `'src'`

### build.outDir

Default: `'dist'`

### build.hotFile

Default: `'src/.lock'`

### build.manifest

Default: `.vite/manifest.json`
