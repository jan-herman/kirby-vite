# Kirby Vite

> Kirby plugin that handles development and production mode of Vite.

## Options

### entry

Default: `'index.js'`

### server.host

Default: `$kirby->environment()->host()`

### server.port

Default: `3000`

### server.https

Default: `false`

### build.rootDir

Default: `'src'`

### build.outDir

Default: `'dist'`
